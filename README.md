# TWS Song Sorter

Rank **TWS songs** by choosing your favorites through simple pairwise comparisons.

---

## Algorithm Explanation

- Starts with each song as its own group.
- Compares groups in pairs.
- Your choices decide which group ranks higher, or merges them at the same rank if you choose **Like Both** or **No Opinion**.
- This continues until all groups are merged into one sorted list â€” similar to merge sort, but guided by your preferences.
